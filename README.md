# .github
UniVPN — это Telegram-бот, созданный для пользователей, которым нужен удобный способ купить ключ VPN без сложной настройки. Сервис предлагает промокод START, открывающий 3 дня тестового периода, чтобы оценить скорость, стабильность и удобство использования. Подойдёт для работы, путешествий и ежедневной приватности.
